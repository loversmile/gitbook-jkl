PyInstaller-3.2

打包成exe可执行文件

  • 使用pyinstaller
安装 pyinstaller
pip install pyinstaller
执行
pyinstaller.exe  -w  --onefile --icon="my.ico" demo.py
--icon         生成exe文件的图标
--onefile     打包成单个文件
-w            不使用控制台输出
pyinstaller 目录
一般来说，pip安装的工具会在Python安装目录的Scripts下面 
E:\Python27\Scripts pyinstaller
pyinstaller 参数
查看参数
pyinstarller -h
